About This Website

This website was created to provide students with an easy-to-understand introduction to AWS and GitHub. It's built using only HTML, CSS, and JavaScript to keep things simple and accessible.

Our Mission

To empower students with knowledge of cloud computing and version control, making complex topics approachable through clear explanations and practical examples.

What You'll Learn

  • Basic concepts of AWS services
  • How to use GitHub for code management
  • Real-world examples tailored for students
  • Links to further learning resources
