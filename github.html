GitHub Basics for Students

GitHub is a platform for version control and collaboration. It uses Git, a tool for tracking changes in code.

What is Version Control?

Version control tracks changes to files over time, allowing you to revert to previous versions and collaborate with others.

Key GitHub Concepts

Repository (Repo)

A folder that contains your project files and their history.

Student Example: Create a repo for your class project to keep all files organized.

Commit

A snapshot of your files at a specific point in time.

Student Example: Save your progress after finishing a feature in your code.

Branch

A separate line of development. You can work on features without affecting the main code.

Student Example: Create a branch to experiment with new ideas without breaking your main project.

Pull Request

A way to propose changes to a repository. Others can review and merge your changes.

Student Example: Submit your assignment changes for your instructor to review.

Getting Started with GitHub

Create a free GitHub account and start by creating your first repository. Use Git commands or the web interface to manage your code.
