# Web-4 Project

This is a simple static website project consisting of multiple HTML pages, along with CSS styling and JavaScript functionality.

## Pages

- **index.html**: The main homepage of the website.
- **about.html**: Information about the project or author.
- **aws.html**: Content related to AWS (Amazon Web Services).
- **github.html**: GitHub-related information or links.
- **resources.html**: Additional resources or links.

## Files

- **styles.css**: Contains the CSS styles for the website.
- **script.js**: Includes JavaScript code for interactive features.

## How to View

Since this is a static website, you can simply open any of the HTML files in a web browser to view the content. For the best experience, start with `index.html`.

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
